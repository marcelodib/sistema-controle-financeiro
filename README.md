# Sistema de Controle Financeiro

A aplicações foi desenvolvida em assembly MIPS e tem como finalidade a gestão de gastos pessoais.

## Funcionalidades

- Controle de gastos.
- Adicionar despesas.
- Excluir despesas.
- Listar todas despesas.
- Listar despesas mensais.
- Listar despesas por categoria.
- Rankear despesas.
